Raccourcis clavier

Vous pouvez utiliser une combinaison de touches (raccourcis clavier) plutôt que d'utiliser votre pointeur afin d'effectuer certaines actions. La liste ci-dessous montre les touches sur lesquelles appuyer simultanément pour obtenir les résulats décrits. Note : Certains raccourcis ne fonctionneront pas avec toutes les Activités.

Combinaison de touches

Description

ctrl + c (shift + ctrl + c dans Terminal)

Copie la sélection

ctrl + v (shift + ctrl + v dans Terminal)

Colle la sélection

ctrl + x (shift + ctrl + x dans Terminal)

Coupe la sélection

ctrl + u lorsque vous utilisez l'Activité Naviguateur

Lance l'Activité Ecrire et affiche le code source de la page web en cours

alt + esc

Quitte une Activité

alt + tab

Passe à l'Activité suivante parmi les Activités en cours

shift + alt + tab

Passe à l'activité précédente parmi les Activités en cours

alt + enter

Bascule en mode plein écran

alt + barre d'espace

Cache/Affiche la barre. Ce raccourci peut ne pas fonctionner dans toutes les Activités.

alt + 1

Enregistre dans le Journal une capture d'écran

ctrl + alt + erase

Redémarre Sugar, l'interface graphique du XO

fn + 1

Vue Voisinage

fn + 2

Vue Groupe

fn + 3

Vue Accueil

fn + 4

Vue Activité

shift + alt + r

Permet de pivoter l'écran

esc + Frame icon key + RightArrow + fn (Les 4 flèches de direction sur le clavier)

Réinitialise le pavé tactile si le pointeur se comporte étrangement lorsque vous utilisez le pavé.

fn + spacebar

Afficher le code source de l'Activité en cours. Ce raccourci peux ne pas fonctionner dans toutes les applications

fn + up arrow

Haut de page

fn + down arrow

Bas de page

fn + left arrow

Début

fn + right arrow

Fin

ctrl + brightness down

Eteint le rétroéclairage - mode noir et blanc

ctrl + brightness up

Active le rétroéclairage - mode couleur

ctrl + volume down

Son au minimum (muet)

ctrl + volume up

Son au maximum
