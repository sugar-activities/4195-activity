Lancer les Activités

Vous pouvez lancer une Activité de quatre manières différentes:

  • En cliquant sur l'icône Activité dans la Vue Accueil.
  • En reprenant une Activité dans la Vue Journal.
  • En rejoignant une Activité partagée dans la Vue Voisinage.
  • En acceptant une invitation par un clic sur l'icône Invitation, dans le Cadre.

Lorsque vous lancez une Activité, son icône clignote au milieu de l'écran tandis que l'Activité se charge.

Lancer depuis la Vue Anneau

Cliquez sur l'icône ou sur une option via le menu déroulant.

Lancer depuis la Vue Liste

Cliquez sur l'icône ou sur une option via le menu déroulant.

Reprendre depuis le Journal

Cliquez sur l'icône ou sur reprendre via le menu déroulant.

Rejoindre une Activité partagée

Cliquez sur l'icône ou sur une option via le menu déroulant.

Rejoindre une Activité en acceptant une invitation

Cliquez Rejoindre dans le menu déroulant du Cadre.
